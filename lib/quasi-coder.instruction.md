# Lib Instructions

This folder will hold:

- Helper scripts
- Support tools
- Repeatitive code that can be reused
- These tools should mostly be utilized by the executables from the `bin` folder of the `sendEmail` tool
- Each repetitive script, helper, or repetitive code blocks that are used in this library must be able to easily integrate or be plugged into an external API i.e. VS Code extension or a GUI library when it is utilized be one of the executables from the `bin` folder
