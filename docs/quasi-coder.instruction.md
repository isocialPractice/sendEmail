# Docs Instructions

## Folder Contents

This folder will hold:

- Documentation pertaining to:
  - Command-line calls with option `-h, --help` or `/?`
  - Use current `sendEmails.txt` as a reference ONLY, but **DO NOT** reference it in the documentation
- The markdown documentation for the `sendEmail` tool

## Make Guide

The file `sendEmail.md` is a variation of the help doc `sendEmails.txt`. It should be used as a guide for creating the command-line tool `sendEmail`. Use it as a guide.
