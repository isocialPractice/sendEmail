# Tests Instructions

This folder will hold:

- Unit tests in the folder `tests/unit`
- Test logs in the folder `tests/logs`
- Tools to run a mock email server in the folder `tests/mock/server(?)=>`
  - Mock server tools that mimic an email server for testing in the folder `tests/mock/server/(?)=>lib`
  - A disposible mock email server for test in the folder `tests/mock/server/mail`
- A test script that will handle the mock email server
- A test script that will handle tests specified with the `--test` option
